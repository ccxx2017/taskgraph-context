# TaskGraph Context

本项目是一个关于"逻辑图与上下文工程"的课程设计与实践项目。

## 项目简介

- 探索逻辑图（Logic Graph）在上下文工程中的应用
- 包含课件、代码示例与实验 Notebook
- 支持上下文打包与 Patch 管理

## 目录结构

```
├── 课程设计/          # 课件与课程资源
├── context_pack_builder.py  # 上下文打包工具
├── logic_graph_task.py      # 逻辑图任务脚本
├── *.ipynb            # Jupyter Notebook 实验文件
├── patch*.json        # Patch 文件
└── graph_state.json   # 图状态数据
```

## 快速开始

1. 确保已安装 Python 3.x 及 Jupyter
2. 安装依赖（如有 requirements.txt）
3. 运行 `lesson.ipynb` 或 `logic_graphic.ipynb` 开始实验
